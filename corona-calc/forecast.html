IMPORTANT NOTICE: This tool is a linear, straight line projection based on current data and trends as reported through John's Hopkins and the CDC. This is NOT a model and should not be interpreted as such. It is simply a status report of where a state or county is right now and where that path will lead if it is continued without changing. In reality, these trends and values will change each day and the ending result will look like a curve. Modeling these outcomes is very complicated and it is not our intention to minimize those efforts. Rather, we are presenting a mathematical, straight line forecasting method, to project how things 'might' end up.
